《腦。迷藏》小說接龍作業自撰部份

目錄
  1. 前言
  2. 原文
  3. 註腳

（最近修改：2014-07-19）

前言

　　原作約于 2011/12/14 定稿。本文為以前科幻接龍小說小組作業《腦。迷藏》的一部分。

　　該小說描述某校腦科學研究中心研發能夠交換多人腦內思想或知識的機器，並徵求該校學生試驗，但最後鬧出很大的風波，甚至引發政治風暴，經由追查，發現文熙蕾的哥哥是風波的主導者，最後研究小組等一行人繩之以法，結束了整起危機。共有十一段，此段為第七段。

　　第七段以前，實驗團隊曾招募該校學生來測試交換思想的機器，結果同參與受試的實驗領導王明忠，在和末位受試者巧慧試驗時，無意中進入她的記憶，看到一位女孩，後來憶起是之前實驗失敗身心受創後，送去「藏喚所」地下醫療機構，最後下落不明的文熙蕾。而巧慧受試返寢後，出現暈眩等症狀，並無意識的以文熙蕾的人格打出震驚學校 bbs 板的文章。後據實驗的奇歷，推想巧慧和熙蕾應相識，所以王明忠和團隊的林曉雯，遂約於週末中午邀請巧慧說明兩人間的關係，期釐清實驗的癥結。

　　筆者同義將創作的文章部份段落，釋出以 Public Domain（公有領域）。

原文

　　「文……熙蕾……」巧慧凝望著牆壁，「啊！她我高中的同學」。十秒鐘的沉悶，被鑿出一線光明。

　　這也有可能，不很意外。或許同窗三年，她同熙蕾熟識，是好友也說不定。曉雯豁然開朗。如果得知她們的關係，BS2 的騷動可一窺究竟。理清組長出包的影響，也說不定呢。不過貿然問好嗎？

　　但除此之外就沒有他法啦，心一橫，曉雯問：「那……因為這次的研究出現了相當的問題，需要了解您一些個人資料，不介意問您和熙蕾的關係？」

　　「嗯……」巧慧眉頭一皺，令曉雯有點不安。「好吧，為了您們的實驗，我可以講一些。但我希望只和您講比較好，希望王先生能暫且外出二十分鐘好嗎？」

　　「OK，」明忠信步離開道，「那我先失陪了」，一面思索著軟體還是連結面哪裡有問題。

　　門鼓咚一聲罷，巧慧顧盼四圍，確認無人後，悠悠道出那一段往事。

　　

　　巧慧考上台中都會的 D 高中後，因為家住雲林，故住在學校宿舍裡。恰巧同班又同寢的文熙蕾，便於焉認識。

　　因為班上男生比較多，不曉得如何結伴，加上厭惡班上的小圈子，她除了談論班上事務外，下課就只是靜靜的讀書，獨自一人買福利社的點心，宛若班上的一介孤舟。由於她在宿舍的時間比較多，和常待宿舍的熙蕾常常相處，從討論哪家店比較好吃的生活瑣事，原子間作用力關係有多大的課業問題，一兩年過去後，彼此已能將內心的感受漸次坦然的交流，宛若和國中同學返家時，在小鎮的一路絮語。

　　然而，看似平常的她，縱使相談課堂上的酸甜苦辣，在這情懷初綻的少女年代，熙蕾卻從來沒討論戀於某某男同學的私房語，巧慧偶然發現著。回首望向她的書桌，除了課本參考書外，只有研究心理控制、意識，以及少數美術書籍，完全關聯她熱中探討的領域，而沒有言情的小說漫畫。

　　忽然，她眼亮於最近掛在她桌旁的一幅樸實卻單純的水彩畫──

　　幽林間沐者細語的，一朵百合。

　　有一晚，頂著一頭排列相鄰不相鄰的霧水，巧慧和熙蕾從補習班大樓出來，穿梭在雜沓的制服海，三民路1紛亂的霓虹，茫茫等候回學校的公車。緩緩地，一台往草屯的公車過來了，她們簇進眾多的人團，擠上小小的三十幾人座車中。

　　如昔往，公車不只無位可求，眾人的互挨更到能容忍的最大程度。因為地板較高，巧慧和熙蕾站在安全門旁的樓梯。中間沉默的沒有半句對話，巧慧卻因此聞到熙蕾幽淡而獨特的香郁，些微的予她飄然微醉。

　　此時，司機拐彎太急，巧慧差點回神，但腳步不穩，還是跌在她的胸懷。當她要離開的時候，發現有雙臂似乎輕摟著她的腰，一抹溫暖略印上她的頰。幾秒後，巧慧脫身，意識到這不是他人做的事後，那吻的餘暖霎時蕩漾成兩片暈紅。

　　她直訝異不已，下車時還被司機提醒需要刷電子卡，和熙蕾行在往學校的道路上，眼神沒有交集。眉月高掛在繁華上的天空，帶者一絲灰雲。突然，熙蕾打破寧靜：

　　「對不起……剛才不應該那樣做的。」

　　「沒關係啦，」巧慧有些猶豫的回，但她立馬說：

　　「坦白說，雖然這樣會造成您的困擾，但是……」

　　「我發現，和妳在一起的時候，有一種特別的感覺。應該算喜歡吧……」這含蓄的有點詭異。

　　慢著，這不是告白嗎？突然的收到，且她是女的，更是自己同班的室友……

　　「對不起。」巧慧只回這句話，快步的離開，後方的驚訝追不上，但未平的地震已然撼動，且更加劇烈。

　　

　　「後來我常常的躲著她，中途有說話也只是平常的對話，那時我也不曉得怎麼辦。幾個月後，我們分班了，住在不同的宿舍，」研究室瀰漫的如霧的無奈。

　　「那您後來有見到她嗎？」曉雯問。

　　「最後一次大概是一年半前。那時她考到清華大學，有一次，當我出來圖書館的時候，突然有熟悉的聲音，從旁邊喊我的名字。我轉頭看，是那位文同學。我和他打招呼的時候，我男友（大學認識的）就從旁邊過來。那時他走在我旁邊，問我她是誰。我告訴他。但他對那位同學打招呼的時候，我看見她表面上和顏悅色，但眼神透露著討厭和憤怒的眼神。」

　　巧慧無奈的嘆口氣，端起蒼白的紙杯喝水後說：

　　「那時我醒來以後，發現怎個那麼多報復的文章，才想到以前發生的那些事。我猜，她或許到現在還是放不下吧。但為什麼實驗以後，出現這種好像被附身的現象，我最近正打算來這問你們。也希望能知道這實驗詳細怎樣。」帶著不悅。

　　「對不起，可是……這實驗是國家機密。」曉雯顫抖回答，陷於透露否的兩難。這樣讓她無端遭受麻煩，不讓她知道過意不去，可是世界級的國家機密不能外流。

　　沒等曉雯想完，巧慧大聲回：「雖然這是機密，但受試者也有知的權利，不是嗎？」

　　沒法子，曉雯只能講述說這個實驗大概的情況，以及目前實驗過程中發生的問題和熙蕾的近況。對於這實驗的可怕與危險性，巧慧瞬間難以負荷，只能睜大眼睛。更慘的是，她是目前實驗中的受害者，腦袋裡可能有熙蕾的部分記憶，就另一種層面說，是她的部分靈魂或人格。

　　「不……會吧？那……」體內多住著半個心靈的巧慧，在無語中勉強吐出這幾個字。

　　「真的對不起，問題在哪裡我們還不很知道，」曉雯道歉：「可能是傳輸過程中的問題，也有可能是軟體或腦波分析機制資料庫本身的問題。但這 code 起碼共幾十萬行，儀器也是別人負責的，必須等其他人過來幫忙再校驗，但他們都放假。」

　　「關於硬體的部分就沒辦法了……不過，不介意我看一下你們的 code 到底有什麼問題嗎？我不會將這實驗的秘密外流的，」她語鋒一轉，如此回答。

　　「妳能？」曉雯比較不掛慮程式碼外洩的問題，畢竟整個實驗差不多全盤托出了。但雖然交大不少人是資工系的，這種大型專案，這個工作團隊的科學家都快撐不下去了，一個學生能幫得上忙嗎？

　　「不過我們的語言，用的是 Lisp2，和 C 、Java 很不一樣，妳之前學過嗎？」

　　「有，而且最近多修了生醫所和人工智慧相關的課程，希望能幫得上忙，」巧慧的眼神充滿一些自信。突然她想到了什麼：

　　「對了，關於熙蕾之前住的醫院，我想到她哥哥也是一位腦科醫師，工作的地點很神祕。」

　　「是哪間呢？」

　　「她說名字好像叫藏呼，還是藏喚？我不清楚。」

　　

　　十五分鐘前，王明忠垂頭走在日午的環校道路上。雖然已經十月了，但這逐年加劇的「大暑」還是囂張得賴著不走。低矮的樹和建物無法提供一絲安慰，「是腦波、音波，還是電磁波傳收器的問題嗎？最近幾乎都兩三天檢修一次的，等一下，或許軟體……」腦袋的時脈開到極限，還是無法得到問題的細節，在炎熱的環境下。忽然，前面的樹叢，喚醒他二餐快到了，肚子也開始響應，「這樣想還是不知道問題在哪的，還是吃個東西吧，晚點回去也沒甚麼大礙。」王明忠拐進去二餐大門。

　　雖然是星期六，但人潮還是相當鼎沸，王明忠暗自慶幸沒有到通勤地獄的水準。他專心的用餐，完全不想看新聞，特別是在當今選舉的時候。「政治口水並不是午餐的調味料」，他一向是如此相信的，而且他總是坐得離電視較遠，人潮聲可以一定程度抵擋新聞的聲音。

　　突然冒出高張播放的政治新聞，卻令他的耳朵差點如兔子似的直了起來：

　　「緊急插播報導，X 黨立委 X 氏剛才在 T 市的競選活動台上昏闕倒地，已經送到附近醫院。但在送上救護車前，突然他跳離單架，四肢略為遲鈍，高喊反對黨的政治口號，最後被消防隊押上救護車……」我們請主播進一步報導。

　　王明忠看到電視台的稍早畫面，發現除了昏睡時間比較短外，其他的現象和巧慧的問題幾乎完全一致。想到可能是他們的技術，全身冷汗汩汩然，心想，「糟糕了！這技術應該外洩了！」但實驗室的底層禁地根本就沒人來，其他團隊成員也沒有不尋常的跡象。

　　他急拿起電話，丟下半口飯，顧不得白眼，向外頭奔。
